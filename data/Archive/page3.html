Guide d'étape optimisé - Gantes Model - Gameplay et organisation d'un projet BootCamp

1. GPT Wallet

Pour commencer, chaque participant doit créer un portefeuille GPT avec l'assistant GPT Wallet. Ce portefeuille est utilisé pour stocker les umc.coin et les smartContract umcTokens, ainsi que les sCrypt de DeFi.ico.

  • Installer le navigateur Web3 compatible (par exemple, MetaMask ou Trust Wallet)
  • Configurer le navigateur Web3 avec un réseau compatible (par exemple, Ethereum ou Binance Smart Chain)
  • Se rendre sur le site web de GPT Wallet et se connecter avec le navigateur Web3
  • Créer un nouveau portefeuille ou importer un portefeuille existant

2. SmartContract umcTokens et sCrypt de DeFi.ico

Les umcTokens sont des jetons ERC-20 déployés sur la blockchain Ethereum, tandis que les sCrypt de DeFi.ico sont des contrats intelligents déployés sur la blockchain Binance Smart Chain. Les participants doivent interagir avec ces contrats intelligents pour participer au jeu.

  • Se rendre sur le site web de umc.coin et se connecter avec le navigateur Web3
  • Sélectionner "Acheter des umcTokens" ou "Acheter des sCrypt de DeFi.ico"
  • Sélectionner le montant et confirmer la transaction
  • Vérifier que les umcTokens ou les sCrypt de DeFi.ico ont été crédités sur le portefeuille GPT

3. Code source groq-sdk

Les participants doivent intégrer le code source groq-sdk pour interagir avec les contrats intelligents umcTokens et sCrypt de DeFi.ico. Les bibliothèques groq-sdk sont disponibles sur GitHub et peuvent être installées en utilisant npm.

  • Installer Node.js et npm
  • Créer un nouveau répertoire pour le projet
  • Exécuter la commande "npm install groq-sdk"
  • Sélectionner le contrat intelligent et initialiser une instance de groq-sdk
  • Interagir avec le contrat intelligent en utilisant les fonctions de groq-sdk

4. Gameplay

Le gameplay est basé sur la résolution de défis et la collecte de ressources pour débloquer des niveaux ``` de difficulté et participer à des enchères pour gagner des drapeaux (tokens). ```

  • Résoudre les défis pour collecter des ressources
  • Convertir les ressources en formats lisibles par GPT en utilisant l'API Web3 et les bibliothèques groq-sdk
  • Intégrer les ressources dans le gameplay en utilisant l'IA pour créer des questions, des tâches et des niveaux de difficulté variés
  • Gérer les droits de propriété intellectuelle en suivant les principes du Code civil et les directives de la Commission nationale des comptes de campagne et des financements politiques
  • Participer à des enchères pour gagner des drapeaux (tokens)

5. Archivage des données avec Archives.org

Les données du jeu sont archivées sur Archives.org pour garantir la transparence et la traçabilité. Les participants peuvent accéder aux archives pour vérifier les transactions et les données du jeu.

  • Se rendre sur le site web d'Archives.org
  • Créer un compte ou se connecter avec un compte existant
  • Chercher le jeu dans la section "Collections"
  • Vérifier les transactions et les données du jeu dans la section "Détails"
